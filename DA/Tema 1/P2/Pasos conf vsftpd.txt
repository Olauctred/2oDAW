Servidor ftp en linux ubuntu (II)
1. Configura el servidor ftp para permitir que los usuarios anónimos puedan conectarse
2. Configura el servidor ftp para no permitir que los usuarios de sistema puedan conectarse
3. Configura el servidor ftp para permitir que los usuarios anónimos puedan subir archivos
4. Sube un archivo al servidor ftp desde tu máquina real
5. Descarga un archivo del servidor ftp a tu máquina real

CONFIGURACIÓN FTP
sudo gedit /etc/vsftpd.conf

1. anonymous_enable=YES
2. #local_enable=YES
3. 	+ write_enable=YES
	+ #anon_upload_enable=NO
	+ anon_other_write_enable=YES #Para sobrescribir archivos o renombrarlos se usa la directiva 
       *+ anon_umask=022 #Máscara de usuario.
	+ anon_mkdir_write_enable=YES #Importante para despliegue de sitios web, ya que contienen árboles de directorios.

	~$ sudo mkdir /srv/ftp/incoming
	~$ sudo chown ftp:ftp /svr/ftp/incoming
# El símbolo <!> se usa en el cliente FTP para indicar que a continuación hay un comando del sistema local.
